## Підсумок для перевірки підрахунків

Нижче наведено контрольні суми, щоб уникати помилок підрахунку:

- **Понеділок**: рядків у таблиці — **9**, унікальних слотів (Час×Дисципліна) — **9**; по групах: A — 5, B — 5, C — 5.
- **Вівторок**: рядків у таблиці — **8**, унікальних слотів (Час×Дисципліна) — **8**; по групах: A — 4, B — 4, C — 4.
- **Середа**: рядків у таблиці — **8**, унікальних слотів (Час×Дисципліна) — **8**; по групах: A — 5, B — 3, C — 5.
- **Четвер**: рядків у таблиці — **8**, унікальних слотів (Час×Дисципліна) — **8**; по групах: A — 5, B — 4, C — 5.
- **П’ятниця**: рядків у таблиці — **8**, унікальних слотів (Час×Дисципліна) — **8**; по групах: A — 5, B — 5, C — 4.

> Примітка: «Всі групи» зараховується окремо до кожної групи A, B, C.
